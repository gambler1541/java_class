.자바의 식별자와 키워드

식별자(Identifier)
	: 자바에서 식별자는 클래스, 인터페이스, 변수, 메소드,
	  배열, 문자열 등을 구분할수 있는 이름입니다.
	  이름은 다른 것과 구별해야 하기 때문에 중복되지 않는 것을
	  사용해야 합니다. 식별자는 길이의 제한이 없고
	  유니코드 문자를 사용해야 하기 때문에 한글도 사용할수 있습니다.
	  대, 소문자를 구분하여 사용하며 식별자를 만드는 몇가지 관례가 있습니다.

	식별자 예

	User_name	O		User name	X
	user_name	O		user name	X
	_User_name	O		#User_name	X
	수선화		O		7백합		X
	$Korea		O		@Seoul		X
	King			O		super		X


	식별자 작성 규칙

		. 유니코드 문자를 사용한다
		. 공백이 없는 한개의 단어로 구성되어야 한다.
		. 문자, 숫자, '_', '$'를 사용할수 있다.
		. 반드시 문자, '_', '$' 로 시작하여야 한다.
		. 길이 제한이 없다.
		. 대, 소문자를 구분한다.
		. 키워드를 사용할수 없다.


키워드(Keyword)
	: 키워드는 자바에서 미리 예약해 놓은 단어를 의미합니다.
	 그래서 키워드를 Reserved Word(예약어)라고 부릅니다.
	 특정한 단어를 미리 예약하여 특별한 용도로 사용하기 위함입니다.
	 자바의 키워드는 모두 소문자 입니다.

					<!-- void는 기본 자료형은 아니지만 기본 자료형처럼 사용됨 -->
	기본 자료형		: void, boolean, byte, char, short, int, long, float, double
					
	변수 유형지정		: transient, volatile
	제어문			: if, else, switch, case, for, while, do, break, continue, return, default

	클래스			: class, interface, abstract, final, extends, implements, this, super, new,
					  instanceof, static

	접근 수정자(지정자)	: public, protected, private
	예외처리			: try, catch, finally, thorw, throws
	패키지			: import, package
	상수				: null, true, false
	메소드 유형		: native, synchronized
					<!-- C언어에서 사용되는 키워드로 자바에서는 사용되지 않음 -->
	사용되지 않는 키워드	: const, goto


변수와 상수 
. 컴퓨터에서 처리하는 값이 변하기 위해서는 그 값을 보관하기 위한 메모리 공간이 필요합니다. 
  변수는 메모리의 특정한 기억공간을 사용합니다.
  그러기 위해서는 메모리의 특정한 위치에 자료를 기억할수 있는 공간을 확보하고
  그 공간에 이름을 부여한 것이 변수입니다.

  . 변수는 이름을 갖는다
  . 변수는 값을 갖는다
  . 변수에 있는 값을 수정할수 있다
  . 변수는 메모리에 위치한다

  => 변수는 자료를 처리하기 위해서 이름이 부여된 기억 공간


 변수 선언
 : 자바는 일반적인 값을 처리하기 위한 기본형 변수와
   클래스와 배열과 같은 참조형을 처리하기 위한 참조형 변수가 있다.

   변수를 사용하기 위해서는 변수를 선언 해야 사용할수 있다.

  형식
 . 자료형 변수 이름

     int        kor		 => 정수형 변수를 저장하기 위해 필요한 기억공간을
				      메모리에 확보하고 그 기억공간을 kor 이라고 선언

     kor = 99;		 => 변수에 값을 할당

.  자료형 변수명 = 초기값;

    int      kor   = 77;  => 변수를 선언함과 동시에 초기화 
  

  변수 선언 순서

 . 변수 이름을 결정합니다.
	: 변수 이름은 변수 작성 규칙에 맞게 정합니다.
	  특히 변수가 처리할 데이터의 성격에 맞는
	  의미있는 이름으로 정합니다.

. 변수의 자료형을 결정합니다.
	: 처리할 데이터의 성격과 크기에 맞는 자료형을 정합니다.

. 변수를 초기화합니다.
	: 대입 연산자(=)를 사용해 선언된 변수에 값을 할당합니다.


변수의 종류

			- 문자형 : char
. 기본형 -		- 정수형 : byte, short, int, long
			- 실수형 : float, double
			- 논리형 : boolean

			- 클래스
. 참조형 -		- 인터페이스
			- 배열
